Grand Stay Hotel - Guest Feedback Form

A elegant, high-end landing page designed for luxury hotel guest engagement. This project demonstrates a polished "premium" UI/UX using core web technologies without the overhead of heavy frameworks.

✨ Key Features

    Premium Aesthetic: A dark-themed, luxury-focused design with high-quality imagery and gold typography.

    Dark/Light Mode Toggle: A custom JavaScript-driven theme switcher to match user preference.

    Interactive Forms: Clean, validated input fields for personal information and guest experience details.

    Smooth Animations: Custom CSS/JS transitions for buttons and interface elements.

    Responsive Layout: Fully optimized for all screen sizes, from mobile phones to large desktop monitors.

📩 Form Handling

This project uses Formspree for reliable, serverless form submissions.

    Seamless Integration: Guests can submit feedback directly from the frontend.

    Instant Notifications: Real-time alerts for hotel management when new feedback is received.

    Spam Protection: Built-in reCAPTCHA and honeypot protection to ensure high-quality data.

🛠️ Built With

    HTML5: Semantic structure for accessibility.

    CSS3: Custom "premium" styling with a focus on dark-themed UI.

    Vanilla JavaScript: Theme switching logic and interactive button animations.

    Formspree: Backend-as-a-service for secure form data processing.

🛠️ Technical Highlights
Advanced Theming System

The project implements a robust dark/light mode toggle using CSS Custom Properties (Variables). By scoping variables to the :root and a .dark-mode class, the entire UI can transition smoothly between themes without redundant code.
